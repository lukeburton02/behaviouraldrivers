# behaviouraldrivers
Evaluating behavioural drivers of changes in social contacts and mobility during COVID-19 in the UK

## Data Sources

- **CoMix social contact data (UK)**: https://zenodo.org/records/6542524
- **Google COVID-19 Community Mobility Reports**: Google LLC. https://www.google.com/covid19/mobility/
- **OxCGRT COVID-19 Government Response Tracker**: https://github.com/OxCGRT/covid-policy-dataset
- **YouGov COVID-19 Behavioural Tracker**: https://github.com/YouGov-Data/covid-19-tracker
